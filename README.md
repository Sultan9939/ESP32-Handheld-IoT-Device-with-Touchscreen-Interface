## Project Overview
This project demonstrates a handheld ESP32-based IoT device with a touchscreen GUI.
The device displays sensor data, allows user interaction via touch input, and manages device states locally.

## Key Features
- ESP32 firmware
- Touchscreen GUI
- Sensor data acquisition (simulated or real)
- State-based device logic
- Clean and maintainable code structure

## Target Use Cases
- Industrial handheld tools
- IoT controllers
- Prototyping of embedded devices with local UI
